# 宠物领养管理系统 java1116

## 项目概述

宠物领养管理系统是一款基于前后端分离架构的信息管理系统，旨在为用户提供一个便捷、高效的宠物领养管理解决方案。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 运行环境
- 系统环境：Windows/Mac

### 开发环境
- IDEA
- JDK 1.8
- Maven
- Mysql

### 技术实现
- 后端：Java、SpringBoot、Mybatis
- 前端：Vue
- 数据库：Mysql
